- Nous sommes constamment en communication

Gestion de Base de Données

Problème : Comment gérer 405 entrées sans hardcoder dans database_helper.dart ?

Solution : Séparation en fichiers de données
```dart
// database_enriched_data.dart
class DatabaseEnrichedData {
  static const coupleQuestionsNew = [
    {'text': '...', 'level': 1},
    // 90 questions
  ];
}

// database_enriched_topics.dart  
class DatabaseEnrichedTopics {
  static const coupleTopicsNew = [
    {'text': '...', 'level': 1, 'duration': 300},
    // 45 sujets
  ];
}

// database_helper.dart
Future<void> _insertInitialData(Database db) async {
  for (var questionData in DatabaseEnrichedData.coupleQuestionsNew) {
    await db.insert('questions', {
      'id': uuid.v4(),
      'category_id': 'couple',
      'text': questionData['text'],
      'level': questionData['level'],
      'is_used': 0,
    });
  }
}
```

Migration de Base de Données

Problème : Comment mettre à jour de 45 à 405 entrées sans perdre les données utilisateur ?

Solution : Système de versioning avec onUpgrade
```dart
Future<Database> _initDB(String filePath) async {
  return await openDatabase(
    path,
    version: 3,  // Incrémenté de 2 à 3
    onCreate: _createDB,
    onUpgrade: _onUpgrade,
  );
}

Future<void> _onUpgrade(Database db, int oldVersion, int newVersion) async {
  if (oldVersion < 3) {
    // Suppression et recréation avec nouvelles données
    await db.execute('DROP TABLE IF EXISTS questions');
    await db.execute('DROP TABLE IF EXISTS topics');
    await _createDB(db, newVersion);
  }
}
```

Service TTS Singleton

Problème : Gérer l'état de lecture vocale à travers plusieurs écrans

Solution : Service singleton avec gestion d'état réactive
```dart
class TtsService {
  static final TtsService instance = TtsService._internal();
  factory TtsService() => instance;
  
  final FlutterTts _flutterTts = FlutterTts();
  
  TtsService._internal() {
    _initialize();
  }
  
  Future<void> _initialize() async {
    await _flutterTts.setLanguage("fr-FR");
    await _flutterTts.setSpeechRate(0.5);
    await _flutterTts.setVolume(1.0);
    await _flutterTts.setPitch(1.0);
  }
  
  Future<void> speak(String text) async {
    await _flutterTts.speak(text);
  }
  
  Future<void> stop() async {
    await _flutterTts.stop();
  }
}
```

Utilisation dans les écrans :
```dart
class QuestionDisplayScreen extends StatefulWidget {
  final _ttsService = TtsService.instance;
  bool _isReading = false;
  
  void _toggleSpeech(String text) async {
    if (_isReading) {
      await _ttsService.stop();
      setState(() => _isReading = false);
    } else {
      await _ttsService.speak(text);
      setState(() => _isReading = true);
    }
  }
}
```

Animations Performantes

Problème : Rotation de roue fluide sur 10 tours complets

Solution : AnimationController avec courbe d'accélération
```dart
class _SpinWheelScreenState extends State<SpinWheelScreen> 
    with SingleTickerProviderStateMixin {
  late AnimationController _controller;
  late Animation<double> _rotationAnimation;
  
  @override
  void initState() {
    super.initState();
    _controller = AnimationController(
      duration: const Duration(seconds: 3),
      vsync: this,
    );
    
    _rotationAnimation = Tween<double>(
      begin: 0,
      end: 10 * 2 * pi,  // 10 tours complets
    ).animate(CurvedAnimation(
      parent: _controller,
      curve: Curves.easeOutCubic,
    ));
  }
  
  void _spinWheel() {
    _controller.forward(from: 0);
  }
}
```

void _startTimer() {

    if (_remainingSeconds > 0) {
      setState(() {
        _remainingSeconds--;

Color _getTimerColor() {
